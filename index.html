My Projects

Each project here reflects a different facet of my creative and technical journey — from SDG-aligned games to decentralized apps.

  • Waroeng Digital Technopreneurship

    A web-based business management platform for traditional Indonesian warung owners (micro-SMEs). Features include collective bulk-buying (Kulakan Kolektif) and a Credit Readiness score to help unbanked entrepreneurs access financing. As team leader, I coordinated the group, set the product direction, and authored the full business plan — covering market analysis, revenue model, and go-to-market strategy.

  • Decentralized Medical Records Blockchain

    Smart Contract Ethereum using metamask — a patient data system that removes centralized authority and gives individuals ownership of their health records. My contribution was all frontend styling: designing and implementing the UI so that a complex blockchain-backed system felt clean and accessible to everyday users.

  • Dijkstra Algorithm Visualizer Algorithms

    An interactive visualization of Dijkstra's shortest-path algorithm implemented with CoppeliaSim. I wrote the code for the algorithm visualization — translating abstract graph-traversal logic into a live, step-by-step visual that makes the pathfinding process intuitive to follow.

  • Game Design Project (SDG) Game Dev

    A game built around Sustainable Development Goals — design with purpose. I was responsible for creating all game art and writing the game design document: defining mechanics, levels, narrative, and how each SDG theme was woven into the player experience.

  • Java Game Game Dev

    A simple but fully playable game built in Java. My role was reviewing and verifying the OOP implementation — ensuring classes, inheritance, encapsulation, and polymorphism were correctly applied, and that the codebase stayed clean and well-structured as the team iterated.

  • DOM Manipulation (JS) Frontend

    A JavaScript project exploring Document Object Model manipulation — dynamic UI updates without a framework, all vanilla JS.

  • FlexBox Price Container Frontend

    A CSS Flexbox exercise building a responsive pricing layout — mastering layout fundamentals that underpin every UI I now design.
